LIST

TYPE

  • ORDERED LIST

    STRUCTURE

    "ol"=To describe that list begins from here

    "li"=To describe the component of list

    EXAMPLE:

    1. HUNGER GAMES
    2. WATCH DOG

  • UNORDERED LISt

    • STRUCTURE

    "ul"=To describe that list begins from here

    "li"=To describe the component of list

    EXAMPLE:

    • HUNGER GAMES
    • WATCH DOG

  • DEFINATION LIST

    • STRUCTURE

    "dl"=To describe that list begins from here

    "dt"=To describe the title of list

    "dd"=To describe the material inside the title.

    EXAMPLE:

    NOUN

    A paticular name of name place or animal.
    VERB:
    In which ACTION is performed.
